# Mod Rewards Inventory

This document catalogs potential loot crate rewards from every mod in the mooStack modpack, organized by value tier. Use this as a reference when expanding loot crate categories or creating new ones.

## Value Tier Definitions

| Tier | Description | Weight Range | Example Items |
|------|-------------|--------------|---------------|
| **Common** | Basic materials, early game | 20-35 | Cobblestone, logs, basic food |
| **Uncommon** | Processed materials, mid-early | 12-20 | Ingots, seeds, basic components |
| **Rare** | Advanced materials, mid game | 6-12 | Alloys, circuits, magical essences |
| **Epic** | High-value items, late-mid | 3-6 | Elite components, rare drops |
| **Legendary** | End-game, extremely valuable | 1-3 | Netherite, ultimate circuits, unique items |

## Usage Notes

1. **Weight Guidelines**: Higher weight = more common drop. Total category weight should sum to ~300-500 for good variety.

2. **Count Ranges**: Use `[min, max]` for variable quantities. Single values for rare items.

3. **Creative Items**: Items marked "(if allowed)" are creative-mode items that may break progression. Consider excluding from normal loot pools.

4. **Mod Availability**: The system gracefully handles missing mods - items simply won't drop if the mod isn't installed.

5. **Balancing**: Consider both raw material value AND progression impact when tiering items.
