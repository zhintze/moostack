## Key Design Decisions

### Combat System Coverage
- Covers stamina, basic attacks, dodging, guarding, and special attacks
- Emphasizes the Skill Tree addon (K key) over random skill books
- Provides skill build recommendations for different playstyles

### Armor Progression
- Gambeson -> Chainmail -> Knight (Armet) -> Gothic/Maximilian
- Includes armor decorations (plumes, etc.)
- Includes horse armor (barding)
- Notes cosmetic slot usage for maintaining appearance

### Antique Legacy Integration
- Minimal showcase (2 quests) per user request
- Raises awareness of ancient armor options
- Directs players to explore bronze/iron materials from Mekanism
